# pdftotext
It is a simple python module that convert pdf to text
# usage
```
Import pdftotextsimple
pdftotextsimple.convert("filename")
```
```
Import pdftotextsimple
pdftotextsimple.convert("D:/ME/example.pdf")
```
# installation 
Stable version: `pip install pdftotextsimple`
